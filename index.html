# Nassa Studio Social media content planning and management tool for agencies. ## Features - 📊 **Studio** — Client dashboard with revenue KPIs - 📋 **Feed** — Visual Instagram feed planner (drag & drop, 4:5 grid) - 📱 **Stories** — Stories management with storyboard builder - 📅 **PED Stories** — Editorial calendar planner for stories - 🗓 **Calendario** — Monthly/weekly calendar view of all content - 👁 **Preview** — Client-facing preview with lightbox ## Deploy on Vercel 1. Push this repo to GitHub 2. Import the repo in [Vercel](https://vercel.com) 3. Vercel auto-detects the static site — click **Deploy** No build step required. Pure HTML/CSS/JS. ## Usage - Add clients via the **Studio** panel - Select client → account → month in the **Feed** tab - Upload media or paste Frame.io / direct links - Export/import the project as JSON to save your work
